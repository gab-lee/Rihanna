# Rihanna

## Project Description

An IOT device that tells you if you should bring an umbrella out.

## How it works

- Time sensitive trigger (hourly, 9am-9pm), device will get weather data from weather API.
- Based on weather data show factual data and visual represesntation

## Display states

Pixelated Tamagotchi vibes

- Cute Visual Gifs
  | - Sunny > 30deg: sweating cat
  | - cloudy: cycling cat
  | - Might rain: brinc umbrella cat
  | - light rain: umbrella open cat
  | - heavy rain: dworning cat and lightning
